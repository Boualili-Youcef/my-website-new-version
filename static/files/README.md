# Fichiers téléchargeables

Ce dossier peut accueillir les fichiers publics du portfolio.

Le rapport technique public de `k8s-sec-lab` est disponible ici:

```text
static/files/k8s-sec-lab-rapport-technique.pdf
```

Pour réactiver un bouton CV, ajoute un PDF propre ici, par exemple:

```text
static/files/CV_Youcef_Boualili.pdf
```

Le lien CV n'est pas affiché tant que le fichier n'est pas fourni.
